# Motd Platform

> MOTD平台（Motd Platform）
> 
> 高质量的服务器状态排行榜，富有拓展性的客户端接口。

## 如何工作

通过序列任务进行数据监测，这将不会占用很多的资源。

完全的前后端分离，以及双向的数据控制。

提供了相关接口，保证了拓展性。

> 在不久的将来，官方将更新完整的后端系统。
